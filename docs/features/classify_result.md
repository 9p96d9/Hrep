# docs/features/classify_result.md — 解析結果画面

**種別: UI仕様（テストなし・自動検証不可）**

---

## 画面構成（上から順）

```
[ヘッダー] HandReporter — 解析結果
[サマリーバー] 総ハンド数 / 青線 / 赤線 / PFのみ
[EV差分表示] ← allin_ev_diffs がある場合のみ
[高難度正解セクション] ← 常に表示（0件でも表示。docs/features/nice_plays.md 参照）
[改善チャンスセクション] ← 常に表示（0件でも表示。docs/features/improve_chances.md 参照）
[タブバー] 青赤線 / ポジション別 / チップ推移
  [タブ①: 青赤線]
    [AI解析結果] ← 解析済みハンドがある場合のみ
    [カテゴリ内訳グリッド] ← クリックでカテゴリフィルター
    [フィルター/ソートバー]
    [ハンド一覧]
  [タブ②: ポジション別]
  [タブ③: チップ推移]
[スティッキーフッター] 解析カート / 3D可視化
```

---

## カテゴリ内訳グリッド

- 各カテゴリをクリック → そのカテゴリのハンドのみ表示（フィルター）
- 同じカテゴリを再クリック → フィルター解除（全て表示）
- アクティブなカテゴリは `cat-active-filter` クラスで視覚的にハイライト

## フィルターバー

| ボタン | 動作 |
|---|---|
| 全て | フィルター解除 |
| 青線 | `data-line="blue"` のハンドのみ |
| 赤線 | `data-line="red"` のハンドのみ（gray/warnは対象外・意図的） |
| AI済 | AI解析結果があるハンドのみ |
| 3BET | `data-3bet="1"` のハンドのみ |

カテゴリフィルターとフィルターバーは排他（どちらか一方のみアクティブ）。

## ハンドカード表示仕様

```
[H番号] [3BETバッジ] [Heroポジション] (Hero) [ヒーロー手札] vs [相手ポジション] [相手手札] [損益]
[アクション履歴（ストリート別・amount_bb付き）]
[GTO数学インライン（hand-gto-math）]
[AIインラインパネル（hai-{hnum}）]
```

- `opp-pos`: 11px太字・グレー背景バッジで表示（視認性確保）
- GTO数学: `📐` アイコン付きで青背景
- AIパネル: 折りたたみ式（クリックで展開）。`detail_street` のストリート別フィールドをストリート順に表示
- スートカラーリング: ♠ #ccc / ♣ #4caf93 / ♥ #e94560 / ♦ #5b9bd5

## ソート

| オプション | 動作 |
|---|---|
| 損益▼ | `data-pl-num` 降順 |
| 損益▲ | `data-pl-num` 昇順 |
| ポジション | UTG→HJ→CO→BTN→SB→BB の配列順 |
| デフォルト | サーバー生成順（カテゴリ別・ハンド番号順） |

同じソートボタンを再クリック → デフォルト順に戻す。
